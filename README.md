# BlazeOptix – Advanced Free Image Optimizer

**A real competitor to paid image optimization plugins.**

BlazeOptix is a **100% free and open-source** WordPress plugin that delivers professional-grade image optimization with **zero limits**, no credits, and no cloud dependency.

### Features
- WebP + AVIF conversion
- Smart quality presets (High / Balanced / Aggressive)
- Background bulk optimization (using Action Scheduler – safe for large libraries)
- Automatic optimization on upload
- One-click restore originals (automatic backups)
- Media Library column with Optimize/Restore buttons
- Clean dashboard with stats
- Lightweight & fast (runs 100% locally)

### Installation
1. Download the latest release or clone the repo.
2. Upload the `blazeoptix` folder to `/wp-content/plugins/`
3. Activate the plugin.
4. Go to **Media → BlazeOptix** to start optimizing.

### Requirements
- WordPress 6.0+
- PHP 7.4+
- GD or Imagick extension (Imagick recommended for better results)
